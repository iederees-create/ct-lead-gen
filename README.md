# CT Lead-Gen Ecosystem — iederees-create

> Automated Cape Town 50-Business Lead Generation & Web Deployment System

## What This Repo Contains

| Folder | Contents |
|---|---|
| `scripts/` | `run_all.ps1` — the master generator + deployer |
| `templates/` | 3 premium CSS/HTML templates (Solar, Wellness, Maintenance) |
| `leads/` | `CT_50_Outreach_Command_Center.csv` — all 50 leads with live URLs + WhatsApp pitch links |
| `clients/` | Local copies of all 50 generated sites (each also has its own GitHub repo) |

## How to Run on a New PC

1. Clone this repo
2. Create `GITHUB_TOKEN.txt` in the root with your GitHub PAT (classic, repo+pages scope)
3. Run the script:
```powershell
powershell -ExecutionPolicy Bypass -File scripts\run_all.ps1
```
